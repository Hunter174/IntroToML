Neural Networks

Welcome to the foundations of Neural Networks! In this section, we'll explore how simple "artificial neurons" can be combined to create powerful models capable of learning complex patterns.

📚 The Perceptron

The Perceptron is one of the simplest types of artificial neurons, designed in the 1950s. It takes several binary inputs, multiplies them by weights, sums them up, and applies an activation function to produce a binary output.

It can solve simple classification tasks where data is **linearly separable**.

📚 Feedforward Neural Networks

Feedforward Neural Networks (FNNs) are the natural extension of the perceptron — layers of neurons stacked together, where information flows forward from input to output without loops.

Each layer transforms the data slightly, helping the network to learn complex representations over multiple layers.

Key Concepts:

  • Input Layer — Raw data input.
  • Hidden Layers — Layers that learn intermediate features.
  • Output Layer — Final prediction or classification.

📚 Activation Functions

Activation functions allow neural networks to model complex, non-linear patterns. Without them, networks would just be stacks of linear functions — not very useful!

Common Activation Functions:

  • Sigmoid — Good for probabilities (outputs between 0 and 1).
  • ReLU (Rectified Linear Unit) — Very popular, fast training for deep networks.
  • Tanh — Output between -1 and 1, used in certain cases.
